# Tipos de Machine Learning "ML"

## 1. ML de Aprendizaje Supervisado "Supervised Learning"

### Qué es:
    El aprendizaje supervisado es un modelo enfocado en el entrenamiento por medio de entradas de información y datos específicos, estos datos e información ya se encuentran enfocados a un tema y en algunos casos ya cuentan con las respuestas correctas permitiendo que aprenda y comprenda cuál es el correcto actuar para poder dar una salida correcta.
    
### Qué conseguimos con esto:
    Gracias a este modelo de ML podemos realizar diferentes predicciones y obtener el resultado esperado.

### Ejemplo de esto:
    podemos encontrarle diferentes usos, tales como la identificación de colores, operaciones matemáticas básicas, clasificación de correos como spam o no spam, predicción de precios en un mercado como en la bolsa o en las viviendas y lo más sorprendente Redes Neuronales el cual busca simular el funcionamiento de un cerebro humano.

## Ventajas y Desventajas

### Ventajas
    1. Tenemos una alta precisión cuando se tienen suficientes datos de calidad.

    2. Nos otorga más flexibilidad que podemos utilizar en diferentes problemas ya sea de forma numérica o como para la clasificación de un sistema o información como entrada.

    3. Podemos automatizar tareas repetitivas, por lo que reducimos la carga de trabajo facilitando las cosas.

    4. Cuentan con bases sólidas las cuales respaldan las decisiones o predicciones que se realicen.

### Desventajas
    1. Para el entrenamiento de depende de una gran carga de datos seleccionados, que pueden ser costosos y difíciles de conseguir dadas las necesidades.

    2. El modelo puede “aprender ruido”, el ruido hace referencia, que si los datos proporcionados son inconsistentes o están mal categorizados, el modelo también aprenderá esos errores.

    3. La escalabilidad del modelo entrenado se puede ver limitada si, los datos de aprendizaje son demasiado complejos o son ingresados en gran cantidad, ya que se saturaría la capacidad de comprensión y aprendizaje de los datos.

    4. No es viable en tiempo real, nos referimos que no es muy útil entornos cambiantes ya que, el modelo se quedaría obsoleto rápidamente, porque los datos son inciertos.

## 2. ML de Aprendizaje No Supervisado "Unsupervised Learning"

### Qué es:
    El aprendizaje No Supervisado es un modelo en el cual no se le entregan entradas de información y datos específicos, el propio modelo se encarga de buscar patrones o agrupaciones con los cuales poder entregar una salida correcta a la solicitada entregada.
    
### Qué conseguimos con esto:
    Gracias a este modelo de ML podemos descubrir estructuras, o relaciones entre los datos sin que el modelo fuera guiado.

### Ejemplo de esto:
    podemos encontrarle diferentes usos, tales como la agrupación de clientes según sus compras, podemos agrupar imágenes similares dentro de la galería de un dispositivo móvil.

## Ventajas y Desventajas

### Ventajas
    1. No necesitamos de datos categorizados lo que nos permite ahorrar tiempo y dinero.

    2. Nos muestra patrones que estaban ocultos y que no son evidentes para uno como humano.

    3. Tiene una gran flexibilidad que podemos usar para agrupar y simplificar datos.

    4. Esta sí se puede adaptar en los diferentes entornos donde sus datos e información cambian constantemente y no da tiempo de categorizarlos.

### Desventajas
    1. Se dificulta la evaluación de la información de salida esto debido a que no utiliza datos categorizados, por lo que no existe una respuesta correcta, solamente una respuesta.

    2. Como se mencionó antes está la posibilidad de aprender ruido, esto por causa de que los datos tienen valores inciertos, entonces el modelo puede generar salidas erróneas.

    3. Tiene una interpretación variante, los patrones encontrados pueden no tener un resultado correcto si no tenemos un contexto específico.

    4. Tiene poca credibilidad o peso que lo respalde ya que el propio algoritmo puede dar resultados distintos según las variables que se le dé o pida.
